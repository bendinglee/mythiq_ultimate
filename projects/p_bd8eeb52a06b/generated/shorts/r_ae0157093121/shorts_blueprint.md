# Shorts Blueprint

## Topic
Turn a long history topic into a viral short with a strong hook

## Hook (0-3s)
Open with the strongest curiosity gap or emotional claim.

## Beat Structure
1. Hook
2. Context
3. Escalation
4. Payoff
5. CTA / loop

## Edit Notes
- fast cuts
- subtitle-first delivery
- motion every 1-2 seconds
- visual resets every beat
- music swell before payoff

## Output Package
- short script
- caption rhythm
- cut notes
- scene prompt ideas
