# Flame Frags Combat Pack

## Goal
Create a Mythiq benchmark pack from 7 Flame Frags videos.

## Outputs
1. Best combat compilation short
2. Best single combat short
3. Review pack of strongest clips across all source videos

## Selection priority
- hook in first 3 seconds
- combat clarity
- outplay / clutch / reversal
- replayability
- montage coherence

## Notes
This pack is the first benchmark set for Mythiq multi-video shorts.
