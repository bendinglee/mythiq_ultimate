# Product Vision — Long-to-Short System

## Goal
Build a professional-grade long-to-short engine that converts long videos into smooth, visually strong, platform-ready short clips.

## What “great” means
A great clip is:
- clearly tied to a real source-video moment
- hooks within the first 1–3 seconds
- has one clean emotional or narrative shape
- feels intentionally framed for vertical
- has readable captions without visual clutter
- is polished enough to post directly

## Core principle
This is not a one-off clip cutter.
This is a self-improving editing engine.

## Operating rule
Every run must leave behind:
- clips
- metadata
- scores
- user choices
- reusable pattern knowledge
