# Quality Rubric

Score each clip from 1-10 on:

1. Hook Strength
2. Source Relevance
3. Story Clarity
4. Pacing Smoothness
5. Visual Framing
6. Caption Readability
7. Emotional Payoff
8. Replayability
9. Professional Polish

## Grade bands
- 9-10 = publish-ready
- 7-8 = strong but needs refinement
- 5-6 = usable but weak
- 1-4 = reject

## Rule
Be harsh.
Most clips should land in 5-8.
Only exceptional clips should reach 9+.
