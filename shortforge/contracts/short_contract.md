# Short Contract

Every generated short must produce:

- title
- description
- transcript
- anchor_text
- score
- relevance
- variant
- source references

## Title rules
- must be understandable alone
- must match the actual clip
- must not overclaim
- should feel native to the niche
- should prefer clarity over spam

## Description rules
- 1 to 3 sentences
- explain what is happening in the clip
- mention the key conflict/problem/reveal
- stay relevant to the actual source
- avoid generic filler

## Montage rules
- montage short must still feel coherent
- use one theme per montage
- avoid random unrelated moments
